여백

여백 적용

margin

바깥여백(=외부여백)

- margin:50px; - 4면(top / right / left / bottom) 적용되는 여백의 값이 모두 같음

- margin-top / margin-right / margin-left / margin-bottom - 4면에 적용되는 여백의 값이 다를 경우

- margin : 50px 100px - 대칭이 되는 두 면의 여백의 값이 같은 경우

padding

안쪽여백(=내부여백)

- padding:50px; - 4면(top / right / left / bottom) 적용되는 여백의 값이 모두 같음

- padding-top / padding-right / padding-left / padding-bottom - 4면에 적용되는 여백의 값이 다를 경우

- padding : 50px 100px - 대칭이 되는 두 면의 여백의 값이 같은 경우
