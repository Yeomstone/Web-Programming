다양한 색상과 배경적용

단일 색상 적용

한가지 색상만을 적용

-색상의 이름 : red, orange...

-색상의 코드 : #FF0000, #FFA500...

-색상의 RGB값 : rgb(255, 0, 0), rgb(255, 165, 0)...

-색상 + 투명도 : rgba(252, 147, 147, a)

a : alpha(투명도) - 0%(0)~100%(1)

배경이미지 적용

.jpg / .png / .gif / .svg / .webP

background-image:url(이미지의 경로);

배경이미지 반복 background-repeat

repeat : 반복, 기본값

no-repeat : 반복하지 않음

repeat-x : 가로 방향으로 반복

repeat-y : 세로 방향으로 반복

배경이미지 크기 background-size

가로 세로 : 수치값으로 직접 입력(단위는 px, %)

contain : 영역에 배경을 채움 (비율유지)

cover : 배경이 영역 전체를 덮음 (비율유지)

배경이미지 위치 background-position

x y : 수치값으로 직접입력 (단위는 px, %)

left / center / right

top / bottom
