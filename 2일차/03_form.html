form - 사용자 입력 양식

웹 페이지의 정보를 다른 페이지로 전송하는 역할

action - 폼 내부의 데이터를 보내는 목적지의 주소

method - 폼 데이터를 전송하는 방법 (GET, POST)

target - 폼 데이터를 전송할 창의 이름 (_self)

name - 폼의 이름

autocomplete

입력 양식의 종류

  • text - 일반 텍스트 입력 :
  • password - 비밀번호 입력 :
  • email - 이메일 입력 :
  • number - 숫자 입력 :
  • date - 날짜 입력 :
  • checkbox - 체크박스 :
  • radio - 라디오 버튼 :
  • search - 검색 :
  • file - 파일 업로드 :
  • submit - 제출 버튼 :
  • reset - 초기화 버튼 :
  • hidden - 숨겨진 입력 :
  • textarea - 여러 줄 입력 :
  • select - 드롭다운 목록 :
  • button - 일반 버튼 :
